# Site-da-gabriely

Pequeno site para declarar o amor por Gabriely.

Como usar:

1. Substitua `assets/images/photo.svg` pela sua foto (mesmo nome ou atualize o `src` em `index.html`).
2. Coloque a música `cidade_vizinha.mp3` em `assets/audio/` ou cole uma URL pública no campo da página; é possível pré-carregar via query string: `https://.../index.html?music=URL_DA_MUSICA`.
3. Personalize o texto em `index.html` (elemento `#message`).

Observação: por motivos de direitos autorais, use apenas músicas que você tenha permissão para publicar. Se preferir, carregue a sua própria cópia do MP3 em `assets/audio/`.

Para publicar via GitHub Pages: vá em Settings → Pages → Source → Branch `main` → `/ (root)`. Depois de ativar, a página ficará disponível em `https://<seu-usuario>.github.io/Site-da-gabriely/`.
